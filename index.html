OSTree Update Service

Welcome to the OSTree update service.

This service provides the latest OSTree repo within an OpenShift cluster. You may also view the OSTree repo directly.

Using this container in a cluster

        $ ssh root@NODE_HOST
        $ ostree remote add --no-gpg-verify os-svc http://os-content.namespace.svc:8080/repo
        $ rpm-ostree rebase -r os-svc:openshift/3.10/x86_64/os

        # wait, SSH back in
        $ openshift version
